3.自己定期备份文件，以免星月舱升级导致不必要的损失
4.所有聊天记录、聊天信息都存在user本地的缓存里，所以手机、电脑、不同浏览器 消息不能互通，只能手动转移备份文件。
5.禁止二传二改商用。
6.这里就像ChatGPT和Gemini一样没有细分的角色卡位，只有不同窗口。不同窗口公用记忆库。
7.一个聊天窗口可生成一次今日记忆，AI自行总结聊天内容放在记忆库，方便记忆的延续。
